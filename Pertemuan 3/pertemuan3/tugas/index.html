Apa itu Version Control?

Version Control Adalah sistem yang mencatat perubahan file atau kode sumber seiring waktu, memungkinkan pelacakan, pemulihan versi sebelumnya, dan kolaborasi tim yang efisien

Apa Perbedaan Git dan Github?

Git : bekerja dalam lokal

Github : bekerja dengan memerlukan koneksi internet

Apa itu HTML?

HTML adalah bahasa standar yang digunakan untuk membuat struktur dasar dari sebuah halaman website
